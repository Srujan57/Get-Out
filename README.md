# Get-Out
Get-Out is a Python escape game where the player navigates through a series of places, attempting to find clues to escape. This project focuses on puzzle structure, and terminal-based gameplay using simple Python code.

Overview:
This is a beginner-friendly Python game that runs in the terminal. The player explores a map represented by characters, enters movement commands, and must find the correct path to escape.

This project demonstrates:
-Game loop structure
-Terminal rendering
-Clean Python fundamentals

Project Structure:
Get-Out/
│
├── main.py        # Main game file
└── README.md

How to Run the Project:
1. Install Python

Check your Python version:
python --version

2. Clone the Repository
git clone https://github.com/Srujan57/Get-Out.git
cd Get-Out

3. Run the Game
python main.py
